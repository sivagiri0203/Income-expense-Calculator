Income & Expense Calculator
This project is a simple web application that helps users keep track of their money. You can add your income and expenses, edit them, delete them, and see your total balance. It also saves your data so that it stays even after you refresh the page.

Features

✔ Add Entries
You can add new income or expense items by entering a description and an amount.
✔ Edit Entries
If you made a mistake, you can edit any entry.
✔ Delete Entries
You can remove items that you no longer want.
✔ Filter
There are three filters:
All
Income
Expense
These filters help you see only what you want.
✔ Totals at the Top
The app shows:
Total Income
Total Expense
Net Balance (Income − Expense)

✔ Local Storage
Your entries are saved in the browser, so they stay even after closing or refreshing the page.
✔ Clean UI
The design is clean and easy to understand.
It works well on both desktop and mobile screens.
---
Technologies Used
HTML — for the layout
CSS — for the design and styling
JavaScript — for adding, editing, deleting, filtering, and saving data
LocalStorage — to keep your entries saved

How to Use
1. Type a description (e.g., "Salary", "Food", "Shopping").
2. Enter the amount.
3. Choose Income or Expense.
4. Click Add.
5. Your entry will appear in the list below.
6. Use Edit or Delete buttons to modify or remove entries.
7. Use the filters (All / Income / Expense) to view selected data.
8. The totals will update automatically.
